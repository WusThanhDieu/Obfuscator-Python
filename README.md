## Obfuscator Python Code
## Features:
- Renaming Function, Classes, Identifiers and Variables names.
- Minifier, Remove Annotations.
- String Encode (zlib+base64).
### Sample:
```python
def hello_world(name):
    print("Hello", name, "!")
hello_world("Alice")
```
